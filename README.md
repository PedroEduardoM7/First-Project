# Simple Translator

This is my first programming project. I created it to practice HTML, CSS, and JavaScript.
I still have a lot of things to improve.

## What it does

Simple: you type a text, choose the language, and it translates it using an API.
Nothing complex, it just works.

## What I learned

This was my first real contact with APIs and the DOM.
I can say I learned a lot by getting things right and wrong.

I had some trouble styling the `<select>`.
I found that browsers try to keep their default style, which made it harder than I expected.

## It's not perfect yet

* The `<select>` didn’t turn out the way I wanted
* The translation API has some limitations since it is free
* The design is still very simple
